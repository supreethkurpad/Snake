SNAKE_C

Recreation of the classic Snake game on terminal with C.

Authors: Team 20, Section-A
Prajwal A
Annirudh A
Supreeth G Kurpad

<==================================================================================================================>
How to execute

The make file is provided. Use command
"make -f make.mk" to get a.out and excecute

<==================================================================================================================>
Pre-requisites

This program uses an external C module called ncurses.
To run this program one must have that module installed on their system.

Here is a guide to install ncurses on Ubuntu
https://www.cyberciti.biz/faq/linux-install-ncurses-library-headers-on-debian-ubuntu-centos-fedora/

Documentation for ncurses
http://www.cs.ukzn.ac.za/~hughm/os/notes/ncurses.html#init
http://www.tldp.org/HOWTO/NCURSES-Programming-HOWTO/helloworld.html 

<==================================================================================================================>

NOTE: Let the terminal be full screen before starting the game
      USE THE KEYBOARD ONLY FOR DIRECTION CHANGE. THE SNAKE MOVES FORWARD ON ITS OWN. TRY NOT TO PRESS THE SAME DIRECTION KEY

<==================================================================================================================>

Game Instructions: Use W,A,S,D to move the snake. As the snake eats the food, it grows longer. Aim is to not touch the boundary or the snake body and gain as much score as possible. +10 score for each food ingested. User will have 3 lives.
